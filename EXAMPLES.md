# نمونه‌های استفاده از بات 📱

## مثال ۱: ثبت مرخصی ساعتی

```
👤 شما: /hourly

🤖 بات: 🕐 ثبت مرخصی ساعتی

لطفاً تاریخ مرخصی را به فرمت زیر وارد کنید:
YYYY-MM-DD

مثال: 2024-03-15

یا /cancel برای لغو

👤 شما: 2024-03-20

🤖 بات: ساعت شروع مرخصی را وارد کنید:
فرمت: HH:MM

مثال: 09:30

👤 شما: 10:00

🤖 بات: ساعت پایان مرخصی را وارد کنید:
فرمت: HH:MM

مثال: 12:30

👤 شما: 14:00

🤖 بات: مدت زمان: 4.00 ساعت

دلیل مرخصی را وارد کنید:
(یا - برای بدون دلیل)

👤 شما: ویزیت پزشک

🤖 بات: ✅ مرخصی ساعتی شما با موفقیت ثبت شد!

📅 تاریخ: 2024-03-20
🕐 از: 10:00
🕐 تا: 14:00
⏱ مدت: 4.00 ساعت
📝 دلیل: ویزیت پزشک

🆔 شناسه: 1
```

## مثال ۳: مشاهده لیست مرخصی‌ها

```
👤 شما: /list

🤖 بات: 📋 *لیست مرخصی‌های شما:*

1. 🕐 ساعتی
   📅 تاریخ: 2024-03-20
   🕐 10:00 - 14:00 (4.00h)
   📝 ویزیت پزشک
   ⏳ وضعیت: pending
   🆔 1

2. 📅 روزانه
   📅 تاریخ: 2024-03-25
   📝 مرخصی استحقاقی
   ⏳ وضعیت: pending
   🆔 2

3. 🕐 ساعتی
   📅 تاریخ: 2024-03-18
   🕐 09:00 - 11:30 (2.50h)
   📝 کار شخصی
   ✅ وضعیت: approved
   🆔 3
```

## مثال ۶: استفاده از دکمه‌های کیبورد

```
👤 شما: /start

🤖 بات: سلام علی عزیز! 👋

به ربات ثبت مرخصی خوش آمدید.

📋 دستورات موجود:
/hourly - ثبت مرخصی ساعتی
/daily - ثبت مرخصی روزانه
/list - مشاهده لیست مرخصی‌ها
/summary - خلاصه مرخصی‌های من
/help - راهنما

برای شروع، یکی از دستورات بالا را انتخاب کنید.

[🕐 مرخصی ساعتی]  [📅 مرخصی روزانه]
[📋 لیست مرخصی‌ها]  [📊 خلاصه]

👤 شما: [کلیک بر روی 🕐 مرخصی ساعتی]

🤖 بات: 🕐 ثبت مرخصی ساعتی...
```

## نکات مهم 💡

### فرمت تاریخ
- ✅ صحیح: `2024-03-15`
- ❌ نادرست: `15/03/2024`, `2024/3/15`, `15-03-2024`

### فرمت ساعت
- ✅ صحیح: `09:30`, `14:00`, `08:15`
- ❌ نادرست: `9:30`, `14`, `14:00:00`

### دلیل مرخصی
- برای وارد کردن دلیل: هر متنی که بخواهید
- برای عدم وارد کردن دلیل: فقط `-` بنویسید

### لغو عملیات
- در هر مرحله از ثبت مرخصی می‌توانید `/cancel` را ارسال کنید

## رفع خطاهای رایج 🔧

### خطای فرمت تاریخ
```
👤 شما: 20/03/2024

🤖 بات: ❌ فرمت تاریخ نامعتبر است. لطفاً مجدداً تلاش کنید.
مثال: 2024-03-15

👤 شما: 2024-03-20

🤖 بات: ساعت شروع مرخصی را وارد کنید...
```

### خطای فرمت ساعت
```
👤 شما: 9:30

🤖 بات: ❌ فرمت ساعت نامعتبر است. لطفاً مجدداً تلاش کنید.
مثال: 09:30

👤 شما: 09:30

🤖 بات: ساعت پایان مرخصی را وارد کنید...
```

### خطای ساعت پایان قبل از شروع
```
👤 شما: 14:00  (ساعت شروع)

🤖 بات: ساعت پایان مرخصی را وارد کنید...

👤 شما: 12:00  (ساعت پایان)

🤖 بات: ❌ ساعت پایان باید بعد از ساعت شروع باشد. لطفاً دوباره ساعت پایان را وارد کنید:

👤 شما: 16:00

🤖 بات: ✅ مرخصی ساعتی شما با موفقیت ثبت شد!...
```
